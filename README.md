# Freelance-Market-Escrow---BlockChain-Project

A blockchain-based freelance marketplace built using Ethereum Smart Contracts and a DApp interface. This platform allows clients and freelancers to interact securely without a middleman using an escrow payment system.

Features
User registration with role-based access (Client, Freelancer, Arbiter)
Freelancer reputation system
Job posting and bidding system
Escrow-based hiring and payments
Work submission and approval workflow
Dispute resolution handled by Arbiter
Dynamic platform fee deduction
MetaMask wallet integration
Real-time blockchain data updates

Tech Stack
Solidity
Ethereum
Truffle
Ganache
Web3.js / Ethers.js
HTML, CSS, JavaScript / React
Smart Contract Functionalities
Register users with wallet addresses
Post and manage freelance jobs
Place bids on open jobs
Hire freelancers with Ether escrow
Release payments after approval
Handle disputes and reputation updates
DApp Features
Role-based dashboard
Job filtering and sorting
MetaMask transaction support
Auto-refresh using blockchain events
Arbiter admin panel for dispute management

Install dependencies
npm install

Start Ganache

Compile contracts
truffle compile

Deploy contracts
truffle migrate

# Start frontend
npm start
/contracts      -> Solidity Smart Contracts
/migrations     -> Deployment files
/src            -> Frontend source code
/test           -> Smart contract tests
